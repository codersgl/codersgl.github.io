训练计划 IV

给定两个以有序链表形式记录的训练计划l1、l2分别记录了两套核心肌群训练项目编号，请合并这两个训练计划，按训练项目编号升序记录于链表并返回。

注意：新链表是通过拼接给定的两个链表的所有节点组成的。

示例 1：

输入：l1 = [1,2,4], l2 = [1,3,4]
输出：[1,1,2,3,4,4]

示例 2：

输入：l1 = [], l2 = []
输出：[]

示例 3：

输入：l1 = [], l2 = [0]
输出：[0]

提示：

$0 <= 链表长度 <= 1000$
